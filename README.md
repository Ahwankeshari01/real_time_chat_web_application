# Installation & Setup
Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Ahwankeshari01/real_time_chat_web_application.git

# Navigate to project directory
cd real_time_chat_web_application

# Install backend dependencies
cd server
npm install

# Start backend server
 npm run dev
#Start the frontend server
npm start
